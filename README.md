# 가장 쉬운 우리동네 공적 마스크 판매처 찾기 - 우리동네 마스크

COVID-19의 확산으로 부족해진 마스크를 보급하는 공적 마스크의 판매처를 찾고, 재고 현황을 조회할 수 있는 애플리케이션 입니다.
